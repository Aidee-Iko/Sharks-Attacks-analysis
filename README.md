# SHARK ATTACKS ANALYSIS

## Summary
This is an interactive dashboard that gives insights into Sharks attacks recorded over the past 100 years, showing trends and patterns in sharks attacks over the years from 1900- 2017.
# Purpose
This analysis helps with the information on what time of the day sharks are most active, what body parts they tend to attack and which species attacks the most.
- Analyze trends over time
- Identify pattern of attack
- Track what time of the day attack is frequent
- Gain insights in gender and distribution of victims
- Observe which country Sharks attacks happens the most

# Tools
- Power BI desktop- For designing interactive dashboard
- Power Query- Used for importing, transforming and loading data
- DAX- For custom measures and calculated columns
- Data Modeling- To organize data accurately
- File Format- .pbix for report and .png for dashboard preview

# Data source
Dataset used for this analysis includes:
- Personal details (Name, Gender, Location, age)
- Day/Date/Time information (Year, Date, Time of Day)

> _Note: This data was gotten from digitaley drive_

# Features & Highlights
 ### Problem Statement
Although Shark attack incidents have been recorded for over 100 years, the data remains largely underutilized for identifying meaningful trends and patterns. 
This project aims to analyze historical data to uncover patterns related to attacks based on location, activity, victim demographics, and shark species in order
to support informed risk assessment and decision-making.

 ### Dashboard Goal
To provide insights 
 - Analyze trends
 - Monitor demographic distribution 
 - Discover attack patterns
